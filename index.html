About me

Nashville native.

I'm a software developer based in Nashville, TN, building full-stack applications across web, mobile, and cloud infrastructure. My work sits at the intersection of data engineering, product development, and AI tooling.

I'm currently developing Kiwi-Finance, a personal finance app with real-time budget tracking, and Athenium, a Nashville data platform that ingests and exposes local data as a commercial product.

I build across the full stack — React Native for mobile, Node.js and Python on the backend, PostgreSQL for data, and AWS for infrastructure. I use Claude Code and Kiro as part of my development workflow.
