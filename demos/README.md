# Interactive Widget Demos

Open `demos/index.html` in your browser to see all options, or click individual demos below:

## Available Demos

1. **Live Data Dashboard** (Current) - `demo-1-dashboard.html`
   - Real-time metrics with animated counters
   - Updating bar charts showing query performance
   - Rows processed and active pipelines

2. **SQL Query Builder** - `demo-2-sql.html`
   - SQL queries typing out with syntax highlighting
   - Result tables populating row-by-row
   - Cycles through JOINs, CTEs, window functions

3. **ETL Pipeline Visualizer** - `demo-3-pipeline.html`
   - Animated flowchart: S3 → Glue → Transform → Redshift
   - Progress bars and status indicators
   - Data flowing between stages

4. **Real-time Metrics Counter** - `demo-4-metrics.html`
   - Speedometer-style gauges
   - Animated counting numbers
   - Pipeline uptime, data processed, system health

5. **VS Code-style Editor** - `demo-5-vscode.html`
   - Live code editor with syntax highlighting
   - Cursor moves around, code writes in real-time
   - Shows Python/SQL snippets

6. **AWS Architecture Diagram** - `demo-8-aws.html`
   - Animated AWS services connecting
   - Data flows between S3, Lambda, Glue, Redshift
   - Icons pulse when active

## How to View

1. Open `demos/index.html` in your browser
2. Click any demo card to see it in action
3. Let me know which one you like best!

## Installation

Just open the HTML files directly in your browser. No build step required.
